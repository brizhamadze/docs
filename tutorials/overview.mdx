# ­ЪЊџ рЃЋрЃўрЃЊрЃћрЃЮ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

HealthTech рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃўрЃА рЃАрЃгрЃљрЃЋрЃџрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃљрЃЏ рЃЊрЃЮрЃюрЃћрЃЊ: **рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃў**, **рЃАрЃљрЃерЃБрЃљрЃџрЃЮ** рЃЊрЃљ **рЃЏрЃЮрЃгрЃўрЃюрЃљрЃЋрЃћ**. 

рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃбрЃБрЃбрЃЮрЃарЃўрЃљрЃџрЃў рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃЏрЃўрЃќрЃюрЃћрЃЉрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃгрЃћрЃЋрЃљрЃЊ рЃЊрЃљ рЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ рЃгрЃўрЃюрЃљ рЃбрЃБрЃбрЃЮрЃарЃўрЃљрЃџрЃћрЃЉрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ.
