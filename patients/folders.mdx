# The Folder Tree

The **Folder Tree** in the left panel is your main navigation tool for patient records. It displays a hierarchical view of all patients and folders within the current Space.

## Creating a New Patient

To start a new record, you need to create a patient folder.

<Steps>
  <Step title="Click the Plus Icon">
    Locate the **(+)** button at the top of the Folder Tree (next to the Settings gear).
  </Step>
  <Step title="Enter Details">
    A new folder will appear. Type the patient's name or ID (e.g., "John Doe" or "PID-12345").
  </Step>
  <Step title="Confirm">
    Press **Enter** to save. The new folder is now ready for documents.
  </Step>
</Steps>

## Folder Actions

Each folder has a context menu for quick management. Hover over any folder row and click the **3-Dots (â‹®)** button.

### Available Actions
- **Upload:** Add files directly to this folder (PDFs, Images, Audio).
- **DICOM:** Upload a full DICOM directory/study.
- **Record Audio:** Immediately start a voice memo for this patient.
- **Workflows:** Trigger automated workflows (e.g., Cardiology Questionnaire).
- **Rename:** Change the folder name.
- **Delete:** Remove the folder.

<Warning>
  **Deleting is Permanent:** Deleting a patient folder will remove all containing documents and DICOM studies. This action cannot be undone.
</Warning>

## Finding Patients

You can quickly find a specific patient without scrolling.
- **Search Bar:** Click the search bar at the top or press `Cmd+K` (Mac) / `Ctrl+K` (Windows).
- **Type Name:** Start typing the patient's name to filter the list instantly.
