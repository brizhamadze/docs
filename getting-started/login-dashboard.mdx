# ­Ъџђ рЃерЃћрЃАрЃЋрЃџрЃљ рЃЊрЃљ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЊрЃљрЃцрЃљ

<Note>
  ­Ъјг **рЃЋрЃўрЃЊрЃћрЃЮ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў рЃЏрЃќрЃљрЃЊрЃЊрЃћрЃЉрЃљ**
  
  рЃћрЃА рЃњрЃЋрЃћрЃарЃЊрЃў рЃЊрЃљрЃћрЃЏрЃљрЃбрЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃЎрЃў рЃерЃћрЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃЋрЃўрЃЊрЃћрЃЮ рЃбрЃБрЃбрЃЮрЃарЃўрЃљрЃџрЃў #1.
</Note>

## рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

HealthTech рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ рЃерЃћрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ рЃЊрЃљ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЊрЃљрЃцрЃўрЃА рЃњрЃљрЃАрЃљрЃфрЃюрЃЮрЃЉрЃљрЃЊ рЃАрЃљрЃГрЃўрЃарЃЮрЃљ:
- рЃћрЃџ-рЃцрЃЮрЃАрЃбрЃўрЃЌ рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ
- OTP рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃљрЃЊрЃљрЃАрЃбрЃБрЃарЃћрЃЉрЃљ
- рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃАрЃўрЃЋрЃарЃфрЃћрЃћрЃЉрЃўрЃА рЃњрЃљрЃфрЃюрЃЮрЃЉрЃљ
- рЃърЃљрЃфрЃўрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃњрЃљрЃњрЃћрЃЉрЃљ

## рЃЊрЃљрЃњрЃћрЃњрЃЏрЃўрЃџрЃў рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃў

рЃЋрЃўрЃЊрЃћрЃЮ рЃњрЃљрЃЎрЃЋрЃћрЃЌрЃўрЃџрЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА:
- РюЁ рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў
- РюЁ рЃАрЃўрЃЋрЃарЃфрЃћрЃћрЃЉрЃўрЃА рЃърЃљрЃюрЃћрЃџрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ
- РюЁ рЃърЃљрЃфрЃўрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃўрЃА рЃЊрЃљ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃўрЃћрЃарЃљрЃарЃЦрЃўрЃљ
- РюЁ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ
